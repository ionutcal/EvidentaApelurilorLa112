# EvidentaApelurilorLa112

O solutie centralizata pentru gestionarea apelurilor de urgenta si coordonarea interventiilor serviciului 112.
Sistemul permite personalului autorizat sa monitorizeze si sa administreze datele in timp real prin intermediul unei interfete web intuitive, avand la baza o structura de baze de date.

<b> Autentificare securizata:</b> Sistem de login dedicat dispecerilor folosind numarul de telefon de serviciu si parola.

<b>Gestionare apeluri:</b> Permite adaugarea, vizualizarea, modificarea si stergerea apelurilor primite.

<b>Coordonare interventii:</b> Asocierea automata sau manuala a interventiilor cu locatii specifice, apeluri active si echipaje disponibile.

<b>Managementul echipajelor:</b> Monitorizarea echipajelor implicate si a tipului de interventie.

Pe partea de Backend am folosit PHP, pentru Frontend HTML si CSS, iar baza de date este facuta in MySQL.

Structura bazei de date: Dispatcher, Apelant, Apel, Interventie, Echipaj, Locatie, Tip_Interventie si Interventie_Echipaj.
